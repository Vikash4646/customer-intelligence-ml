# customer-intelligence-ml

## Customer Intelligence using Machine Learning

## Problem Statement
Businesses often collect large amounts of customer data but struggle to extract meaningful insights.
This project focuses on understanding customer behavior by grouping customers into meaningful segments using machine learning.

## Objective
- Analyze customer data
- Identify patterns and customer segments
- Build a foundation for customer behavior prediction using ML techniques

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn
